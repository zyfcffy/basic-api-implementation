根据课堂上的demo，完成下面需求
1. 提供获取某一条热搜事件的接口
2. 提供能够根据起始参数，获取对应范围内的热搜事件列表的接口
3. 提供添加热搜事件的接口（事件包含两个字段：事件名称和关键字）
4. 提供修改某条热搜事件的接口（demo没有展示，请大家自己完成）

         如果两个字段都传了，那么都进行修改
         
* 需求5： 通过参数传递的序号，删除列表中对应的某条事件数据
